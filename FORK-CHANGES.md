# Fork Changes vs Upstream

This file documents all fork-specific changes relative to [`upstream/main`](https://github.com/pingdotgg/t3code) (`upstream/main`). The fork lives at [`imabdulazeez/t3code`](https://github.com/imabdulazeez/t3code) (`origin`) and is periodically synced with upstream, re-applying these additions on top.

> **IMPORTANT:** When making any deviating changes to this fork, update this table in the same change. Add a new row (or amend an existing one) describing the change, the files touched, and the key changes. Amending an existing row, and combining similar changes is preferred whenever possible, rather than appending rows unnecessarily. Keeping this table current is mandatory. Do not add upstream changes to this file unless the incoming upstream changes had to be changed/modified to support the fork changes. Simple upstream changes are not required to be added.
